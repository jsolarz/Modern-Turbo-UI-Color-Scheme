# Change Log

All notable changes to the Modern Turbo Pascal UI extension will be documented in this file.

## [0.1.0] - 2024-12-25

### Added
- Initial release with two theme variants
- Nord Turbo Pascal Modern theme
- Borland Turbo Pascal Original theme
- Full syntax highlighting support for all languages
- High contrast design for readability
- Professional UI color scheme

### Features
- Dark navy/blue backgrounds
- High contrast syntax highlighting
- Consistent color palette
- Support for all VS Code features (IntelliSense, Git, etc.)
